css

  • 概述

    层叠样式表，可以用来美化html页面

  • div、span、p标签

    这三个标签都可以看做是容器，可以包裹其他的html内容，然后统一设置样式

    • div：独占一行
    • span：多个span会在一行并排排列，直到一行排满才会换行
    • p：段落标签，独占一行，但是在行的上下会留一些空白
    • 元素的类型：
      • 块级元素：默认情况下独占一行的元素，例如div、p、h1-h6、br
      • 行内元素：默认情况下多个元素处在同一行，例如span、font

  • 在html中引入css

    • 直接在对应的标签上添加style属性

       可以通过style属性为标签设置样式

       这种方式不推荐大量使用，会造成页面结构的混乱，不利于维护与扩展
    • 通过style标签引入css

       在head中添加style标签，将所有的css代码写入此标签中

       这种方式将css在style标签中统一管理，初步实现了html代码和css代码的分离
    • 引入外部的css文件

       通过head标签下的link标签引入外部的css文件

       这种方式真正实现了html与css的代码分离

  • 选择器

    选择器用来选中需要修饰的标签

    • 基本选择器
      • 标签名选择器：

         通过标签名选中元素

         格式：标签名{css样式}
      • 类选择器：

         通过标签上的class属性为标签设置所属的类，所有class值相同的为一类，可以通过类选择器选中同一类的元素

         格式：.类名{}
      • id选择器：

         通过标签上的id属性为标签设置编号，id是唯一的，通过id值选中对应元素

         格式：#id值{}
    • 扩展选择器
      • 后代选择器

         在父选择器中，再选中指定的后代元素

         格式：父选择器 后代选择器{}
      • 子元素选择器

         在父选择器中，再选中指定的子元素

         格式：父选择器>子元素选择器{}
      • 分组选择器

         可以将多个选择器选中的元素一起修饰

         格式：选择器1，选择器2，…，选择器n{}
      • 属性选择器

         可以通过属性条件选中元素

         格式：选择器[属性条件]{}
      • 相邻兄弟选择器

         如果两个元素具有相同的父元素，并且是挨着的，这两个元素就是相邻兄弟，可以根据前面的A元素选中后面的B元素

         格式：大哥+小弟{}
      • 伪元素选择器

         伪元素选择器不仅选中元素，还包括它的状态

         状态分为如下几种：

        :link   表示元素未被点击时的状态

        :hover   表示鼠标移入时状态

        :active   表示元素被点击时状态

        :visited   表示元素被点击后的状态

  • 边框、边距

    • 边框：

      border-可以分别设置上下左右的边框，也可以一起设置
      • 外边距：

        元素和父元素之间的距离

        margin-可以分别设置上下左右的外边距，也可以一起设置，行内元素设置上下外边距无效
      • 内边距：

        元素和子元素之间的距离

        padding-可以分别设置上下左右的内边距，也可以一起设置

  • 常用属性

    元素类型补充
块级元素：独占一行，块级元素可以设置所有属性，块级元素可以包含任何元素
行内元素：同行显示，不可以设置宽、高和上下外边距，只能包含行内元素
display属性
block：块级元素默认值
inline：行内元素的默认值
inline-block：结合了行内元素与块级元素的特点，可以同行显示，也可以设置宽高
none：隐藏元素
text-align:对齐方式
font-size：字体大小
font-weight：字体加粗
font-family：字体样式
color：字体颜色
text-decoration：下划线
background-color：背景颜色
background-image：背景图片
background-repeat：背景图片的铺排方式
background-position：背景图片位置
width：宽
height：高
