JavaScript-学习笔记JavaScript

javaScript

  • JavaScript是脚本语言，无需编译 。JavaScript基于对象，弱类型
  • js优点
    • 交互性：可以与用户进行动态交互
    • 安全性：js只能在浏览器内运行，不能访问本地硬盘或其他资源
    • 跨平台：只要有浏览器即可，与平台无关
  • 在html中引入js
    • 在html中书写js代码

      在html中添加script标签，在此标签中书写js代码
    • 引入外部的js文件

      可以将js代码写到一个独立的js文件中，通过script标签引入此文件

      注意：引入外部js文件的script标签中不要添加其他js代码

js语法:

  • 注释

    • 单行注释：//
    • 多行注释：/* */

  • 数据类型

    • 基本数据类型
      • 数值类型（number）

        js中所有的数值底层都是浮点型，浮点型和整型之间会自动转换

        如 2.4+3.6=6

        特殊值：Infinity 正无穷 -Infinity 负无穷

        NaN（not a number）非数字

        可以通过isNaN(xxx)判断是否为非数字，如isNaN(3)–false
    • 字符串类型（string）

      js中，字符串是基本数据类型，可以用单引号或者双引号引起来

      如：var str1 = ‘abc’; var str2 = “def”;
    • 布尔类型（boolean）

      值为true或false
    • undefined类型

      undefined值只有一个，就是undefined，表示变量未定义，如声明了一个变量，没有初始化，这个变量就是undefined

      如：var a; 此时a的值为undefined
    • null类型

      null表示空，一般作为返回值使用
    • 复杂数据类型：

      数组、对象等都是复杂数据类型

  • 变量的定义

    在js中，所有的变量都是通过var关键字声明的，变量可以指向任意类型

    如：var str = “abc”;   var i = 100;

  • 运算符

    js中的运算符和java中的大致相同

    • 不同之处：

      比较运算符除了”==”，还有”===”
      • ==：在比较时，底层会先进行数据类型转换，然后再比较
      • ===：为严格相等，比较时如果两端数据类型不一致，则直接返回false

  • typeof：用来返回数据类型

  • delete：用来删除数组的元素或对象的属性等

  • 语句

    • js语句和java基本一致
    • if条件语句
    • switch语句，也支持字符串类型
    • 循环语句

        while(){}   do{}while()   for(){}
    • 注意：js不支持增强for循环

  • 函数

    • 通过function关键字定义函数

      function 函数名(形参列表){

      函数体

      }
    • 调用函数：函数名(实参列表)

  • 数组

    • 创建数组

       var arr1 = new Array(10);//创建10个长度的空数组

       var arr2 = [“abc”,18,true];//创建给定初始值的数组

      注意：
      • 数组中可以存放任意类型的数据
      • 如果某个位置没有元素，该位置的值为undefined
      • 数组的长度可以任意改变

  • js的内置对象（API）

    • String对象

      创建：

      var str = new String(“abc”)

      var str2 = “def”

    • 常用的属性和方法

      length          字符串的长度
str.match(regexp) --在字符串中查找匹配正则的值
str.replace(regexp,replaceText) --根据正则匹配并替换
str.search(regexp) --查找第一次匹配到正则的位置

    • RegExp对象（正则）
      创建：

      var reg1 = new RegExp(“”,””);

      var reg2 = /xxx/[ig];

      标识符：g-global 全局查找   i-ignoreCase忽略大小写

      ^-表示以…开头   $-表示以…结尾

      方法：reg.test(str)检查字符串是否匹配正则，匹配为true

    • Array对象

      sort() 对数组的元素进行排序

    • Math对象

      Math可以直接使用，无需创建

      • Math.ceil(3.14) 向上取整
      • Math.floor(3.14) 向下取整
      • Math.round(3.14) 四舍五入
      • Math.random() 返回0-1之间的随机数

    • 全局对象

      全局对象没有语法，直接调用方法即可

      • isNaN(value) 判断是否为非数字
      • eval(codeString) 将字符串当做代码执行

  • 自定义对象

    var person = {
"name" : "zhangsan",
"age" : 18,
"friends" : ["lisi","wangwu","zhangsanfeng"],
"gf" : {
"name" : "zhaoliu",
"age" : 48
}
}

      person对象也是符合json格式

     对象中取值：person.name 或者 person[“age”]
